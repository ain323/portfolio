## 개요

개인 프로젝트와 기술 역량을 효과적으로 소개하기 위해 제작한 포트폴리오 웹사이트이다.<br>
프로젝트 소개, 기술 스택, 프로젝트 경험 등을 한눈에 확인할 수 있도록 구성하였으며,<br>
반응형 레이아웃을 적용하여 다양한 디바이스 환경에서도 최적의 사용자 경험을 제공하도록 구현하였다.

## 주요 기능

- Media Query를 활용한 반응형 레이아웃 구현
- 프로젝트 경험을 한눈에 확인할 수 있도록 구성
- 웹사이트 내에서 직접 연락 가능
- 핵심 정보를 빠르게 확인할 수 있는 화면 제공
